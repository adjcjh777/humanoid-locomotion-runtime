## 4. What is actually new enough（可保留贡献措辞）

1. **A memory-value diagnostic for recovery**：把记忆贡献分解为 decision-flip 率与 flip 条件下的 matched-seed 恢复增益，给出"记忆有条件有用"的可量化判据（headline）。
2. **A typed, seeded, reproducible failure-injection protocol** 专为 frozen humanoid locomotion controller，与 recovery 动作集对齐，支持 per-type 归因。
3. **A supervisory recovery formulation** 将记忆限定在高层动作选择、与底层步态解耦，作为研究"记忆在何层级起作用"的受控测试床。

## 5. What must be removed or softened

- 删除/弱化"first supervisory recovery policy"与"novel 8-action space"措辞，改为"a typed action set"。
- "memory helps"不可作普适结论，必须前置条件（long-horizon / cumulative / degradation）。

## 6. Minimum evidence package

- **Matched-seed counterfactual**：同一 seed 同一注入下 memory-on vs memory-off，报告 decision-flip 率与恢复成功 delta，带 95% CI 与配对显著性检验（McNemar / bootstrap）。
- **Per-failure-type 分解**：transient / cumulative / degradation / long-horizon 分层，验证条件性结论而非平均效应。
- **Memory 组件消融**：event trace、body-state trend、language context 单独与组合，定位有效成分。
- **Negative control**：随机/打乱 memory，证明增益来自信息而非动作多样性。
- **多 seed × 多任务**统计（≥若干 seed），避免单 run 结论。
